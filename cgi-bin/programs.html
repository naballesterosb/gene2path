Programs

Dictionary
The sequences for a selected group of genes (list/Accession/Number) can be obtained in a “fasta” format using the getFASTA program. The gene list contains the “Accession Numbers” from GenBank.

Microarray analysis
The programme automates the analysis of the data generated by the microarray scanner. In our case, values from the "g" signal process file in ".txt" format were used. Among the programme's options, the user can choose whether the data files correspond to control or experimental groups, and the level of gene expression to be visualized on the web site can be established.

Orthology
By searching three NCBI databases, the "Orthology" program provides a tool to identify a homologous gene in another species. This program requires only two parameters to be entered: the "accession number" corresponding to the identification code of the gene selected; and the "species" for which the search is being performed.

Pathways
To obtain KEGG lists for the gene identified in each species (according to the gene ID). Thus, the program searches for the respective gene from a given species in the KEGG_PATHWAY database (http://www.genome.jp/kegg/pathway.html), obtaining and examining the KEGG lists (keggData) to define their ID, url, description name and additional data.

Inverse Search
This step is that of finding all the genes involved in a particular KEGG pathway that are identified by a keggID label. All the data related to the selected KEGG are obtained from the KEGG2 database (http://www.genome.jp/kegg/kegg2.html) and the data obtained are saved in a file that contains: the KEGGData image, the module information for which the conversion codes of the image-gene number are stored; and the respective sequences in FASTA format.
