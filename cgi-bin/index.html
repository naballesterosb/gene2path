Gene2Path: a software to find orthologous genes and to generate KEGG pathways in uncommon species.

It is a package designed:
  1.     To perform visual mapping of these data
  2.     To visualize and analyze orthologous genes
  3.     To visualize pathway datasets, such as KEGG.

Users can search for orthologous genes between particular species and define their percentage similarity through the NCBI website.

The input files are the original data from microarray scanning (txt extension). We recommend saving the results or output files to your personal PC or laptop so that they can be used later by all the applications, and by other software like BLAST2GO, REACTOME, CYTOSCAPE or the software of your choice.

The whole code is written in python and bash, and it works in any browser.

Nowadays this web interface for gene2Path provides a low speed in data processing. If you have several number of gene or big size files, we recommend to use the bash and python script through unix commands.  The complete source code of gene2Path tools can be download from: gene2path-20130123.tar.gz
